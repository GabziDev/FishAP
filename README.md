# FishAP

FishAP est un point d'accès WLAN malveillant conçu pour réaliser des tests de sécurité WLAN.

### Doxygen Doc
Debian/Ubuntu:
- `sudo apt-get install doxygen`
- `doxygen doxygen/Doxyfile`
- Page : doc/html/index.html

### Build Project
Debian/Ubuntu
- `mkdir build`
- `cd build`
- `cmake ..`
- `make`
- `./FishAP`
